# 🎓 SkillChain — Skills On-Chain Platform

> Decentralized Skills & Credentials Platform built on Polygon Blockchain
> Built for Web3 Hackathon 2026 by Team ChainForge

## 🌟 Overview

SkillChain is a decentralized platform that enables issuing, storing, and verifying skill credentials on the Polygon blockchain. Credentials are tamper-proof, trustless, and portable.

## 🚀 Features

- 📜 **Issue Certificates** — Issue skill credentials as on-chain records
- 🔍 **Verify Credentials** — Instant trustless verification
- 🏆 **My Certificates** — View all credentials for any wallet
- 🔐 **Tamper-Proof** — Immutable blockchain storage
- 🌐 **Portable** — Share anywhere, verify everywhere

## 🛠️ Tech Stack

- **Blockchain:** Polygon (Amoy Testnet)
- **Smart Contracts:** Solidity + Hardhat
- **Frontend:** React + TypeScript + Ethers.js
- **Backend:** Node.js + Express
